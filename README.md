# Mapwise Task

Мобільний застосунок на `Expo + React Native` з роутингом через `expo-router`.

## Технології

- `expo` `react-native` `react`
- `expo-router`
- `mobx` `mobx-react-lite`
- `typescript`
- `eslint` (`expo lint`)

## Структура проєкту

```text
app/                  # екрани та маршрути (expo-router)
components/           # перевикористовувані UI-компоненти
constants/            # константи (кольори тощо)
hooks/                # кастомні хуки
stores/               # стани/стори
assets/               # статичні ресурси (зображення)
```

## Швидкий старт

```bash
npm install
npx expo start
```

Після запуску:

- встановіть `Expo Go` на телефон;
- відскануйте QR-код з терміналу;
- застосунок відкриється на мобілці в Expo Go.
